Biliboard UI 4.0 为Biliboard视频频道制作的用户界面设计

您好！

本项目旨在探索数字边界的视觉表现形式。我们通过生成式算法（Generative Art）重构了传统的品牌符号，使其在动态媒介中呈现出流动的生命力。

通过与 Nike Lab 的深度合作，我们将这一视觉系统应用于线上发布会及线下装置中。设计的核心理念在于“控制与失控”的平衡——即在严谨的网格系统中引入随机噪点。
