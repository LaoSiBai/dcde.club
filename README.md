# DCDE.CLUB - Design Lab Portfolio

DCDE 是一个专注于视觉包装 (Visual Packaging) 与 3D 沉浸式体验的个人设计实验室。

## 🚀 项目概览

本项目采用原生开发方案，通过 3D 球体排列展示设计项目，并在点击详情时提供电影级的沉浸式缩放体验。

- **官方网站**: [dcde.club](https://dcde.club)
- **GitHub**: [LaoSiBai/dcde.club](https://github.com/LaoSiBai/dcde.club)

## 🛠️ 技术栈

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Graphics**: [Three.js](https://threejs.org/) (用于 3D 核心交互)
- **Typography**: Space Grotesk (En) + Noto Sans SC (Cn)
- **Deployment**: Vercel/Webhooks

## ✨ 核心特性

- **Immersive Focus Mode**: 点击卡片平滑进入 3D 聚焦视角，背景实时模糊并浮现项目详情。
- **Ultra-Compact Typography**: 经过极简主义改良的 4 级字号体系，配合负字间距 (`-0.01em` ~ `-0.03em`)，呈现高端时尚质感。
- **Style Guide Driven**: 内置视觉规范页面，确保全站设计高度统一。
- **Mobile Optimized**: 完整的移动端手势支持与自适应排版。
